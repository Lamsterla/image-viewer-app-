# image-viewer-app-

A custom-built desktop image viewer application made with Python and Tkinter.

## ✨ Features

- Simple, clean GUI built with Tkinter
- Browse and view images from your local file system
- Uses Python's built-in `os` and `sys` modules for file/directory handling and system-level operations
- Lightweight — minimal external dependencies

## 🛠️ Tech Stack

| Layer | Technology |
|---|---|
| Language | Python |
| GUI | Tkinter (Python's built-in GUI toolkit) |
| System/File Handling | Python standard library — `os`, `sys` |

## 📁 Project Structure

```
├── image.py      # Main image viewer application
└── README.md
```

## 🚀 Getting Started

### Prerequisites
- Python 3.x installed (Tkinter ships with most standard Python installations)

### Run Locally
```bash
python image.py
```
